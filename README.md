# cmr

# CRM Project with AI Assistant 🤖

This project is a modern Customer Relationship Management (CRM) system with a GitHub-style UI and an AI-powered assistant chatbot.

## 🚀 Technologies

- ⚛️ React (Vite)
- 🧩 CSS Modules
- 🧠 Chatbot interface with AI (in progress)
- 🎨 Dark theme inspired by GitHub
